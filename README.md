## Getting Started

1. Setup project:

  ```bash
  git clone https://github.com/datsfilipe/datsfilipe.dev.git
  cd datsfilipe.dev
  pnpm install
  ```

2. And start the development server:

  ```bash
  pnpm dev
  ```
